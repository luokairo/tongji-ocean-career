::: warning 数据说明
- 表格中数据为**示例占位**，将由「企业调研与资源对接组」从院系就业办整理汇总后替换为真实数据。
- 所有数据**仅聚合呈现**，不涉及个人信息。
- 单位名称**按类别归并**，不直接列出个人去向。
:::

## 代表性单位（按类别）

### 央企 / 国企

中船集团及下属研究所、中海油、中交建、中铁建、中国电建、中国船舶工业系统工程研究院、中国移动 / 联通海洋通信部……

### 研究所 / 事业单位

中科院（海洋所、深海所、上海高研院、声学所等）、自然资源部各海区分局、上海/浙江/广东海洋监测中心、各高校海洋研究院……

### 民营企业

涉海机器人/装备初创、海洋大数据/遥感公司、海上风电运维公司、传统船舶设计公司、海洋环境监测公司……

### 海外深造

主流方向 = Scripps（UCSD）、Oregon State、Texas A&M、Imperial、Southampton、TU Delft、东京大学、香港科技大学（广州）等。

## 行业趋势观察（基于近 3 届）

::: tip 趋势
- **海上风电相关岗位**自 2022 起明显增长，覆盖设计、运维、监测
- **海洋 AI / 大数据**类初创吸纳了一批跨学科同学
- **传统船舶设计**岗位需求稳定，央企留用率高
- **海外深造**比例略下降，部分受签证与就业环境影响
:::
