# 求职时间线

汇总同济海洋方向**本科 / 硕士 / 博士**不同赛道的全年关键时间节点。请根据自己的目标提前 6–12 个月开始准备。

## 本科生（应届）

| 时间 | 节点 | 备注 |
| --- | --- | --- |
| 大三 暑期 | **暑期实习**（互联网/金融岗有效率高） | 央企/国企实习一般通过校园渠道 |
| 大四 7–8 月 | 准备简历、套磁、模拟笔面 | 选定 3 大主线（企业/选调/读研） |
| 大四 9–11 月 | **秋招**（民企 8 月已开始，央企 9 月集中） | 投递黄金期 |
| 大四 10–12 月 | **公务员 / 选调** 公告、报名、笔试 | 各省时间不同 |
| 大四 1–3 月 | 体检 / 政审 / 春招 | 春招补位 |
| 大四 4–6 月 | 三方签订 / 论文 / 离校 | 锁 offer 阶段 |

## 硕士生（应届）

| 时间 | 节点 | 备注 |
| --- | --- | --- |
| 研一 暑期 | 实验室项目 + 第一段实习 | 关键基础积累期 |
| 研二 暑期 | **暑期核心实习**（决定留用） | 央企/研究所留用率高 |
| 研二 9–12 月 | **提前批 / 秋招**（部分央企已开） | 投递不要等到大半年后 |
| 研二 1–3 月 | 选调笔试 / 春招 | 选调集中在这个区间 |
| 研三 9–11 月 | **秋招主力期** | 多线投递，签三方 |
| 研三 12 月 – 次年 6 月 | 论文 + 答辩 + 入职 | 平衡好节奏 |

## 博士生（应届）

| 时间 | 节点 | 备注 |
| --- | --- | --- |
| 博三 起 | 与导师沟通**毕业去向意向** | 决定走科研/工程/政府路线 |
| 博四 9–12 月 | 投递目标高校 / 研究所 | 一般包括学术汇报 |
| 博四 1–3 月 | 博士后申请（如选择过渡） | 中流砥柱备选 |
| 答辩前 6 个月 | 体检 / 政审 / 入职手续 | 提前安排 |

## 关键时间窗口（合并视图）

```
 1月 ━━━━━ 春节 / 选调集中面试
 2月 ━━━━━ 春招启动
 3月 ━━━━━ 春招 / 出国录取
 4月 ━━━━━ 春招冲刺
 5月 ━━━━━ 入职体检 / 论文
 6月 ━━━━━ 答辩 / 离校
 7月 ━━━━━ 入职过渡
 8月 ━━━━━ 民企秋招启动 / 提前批
 9月 ━━━━━ ☀️ 秋招主战场（央企+国企）
10月 ━━━━━ 公务员 / 选调公告
11月 ━━━━━ 国考笔试 / 校招高峰
12月 ━━━━━ 笔面试集中 / 签 offer
```

## 常见误区

::: warning 关于"等准备好再投"
**投递永远不会觉得准备好**。建议把第一份简历当"练手"投出去，反馈才是最好的优化器。
:::

::: warning 关于"秋招签了就是终点"
大量同学会在春招重新选择。**三方协议有违约成本但不是不可解**，关键还是早做规划。
:::

::: warning 关于"考研失败再找工作"
"考研先，找工作后"的模式风险大。建议**研招 + 求职双线并行**，11 月前完成主战场布局。
:::
