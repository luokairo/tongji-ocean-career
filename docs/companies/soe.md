## 中国船舶集团有限公司第七〇八研究所（示例）

> **行业方向**：船舶与海洋工程设计 · **所在地**：上海临港

| 项目 | 内容 |
| --- | --- |
| 主要岗位 | 船舶总体设计、流体力学、结构设计、海洋装备研发 |
| 学历要求 | 硕士及以上（部分核心岗位要求博士） |
| 同济对口专业 | 船舶与海洋工程、流体力学、结构工程、海洋技术 |
| 招聘流程 | 9 月网申 → 笔试（数学+专业）→ 一面（专业）→ 二面（综合）→ 体检 → offer |
| 福利 | 落户、住房补贴、年终奖、博后流动站 |
| 投递渠道 | 官网 / 智联校招 / 院系内推群 |
| 信息时效 | 2026-04（示例数据） |

**面试准备建议**

- 流体力学基本方程的物理意义与工程应用
- 一个完整科研项目从问题定义到论文产出的全流程
- 关注国内大型船舶/海洋装备项目动态
