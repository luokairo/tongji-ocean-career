## 临港某海洋装备公司（示例脱敏）

> **行业方向**：水下机器人 / AUV / ROV · **所在地**：上海临港

| 项目 | 内容 |
| --- | --- |
| 主要岗位 | 控制算法、机械设计、嵌入式、海试工程师 |
| 学历要求 | 本科及以上，硕士优先 |
| 同济对口专业 | 海洋技术、控制科学、机械、计算机 |
| 招聘流程 | 简历筛选 → 技术面（1–2 轮）→ HR 面 → offer |
| 实习机会 | 提供 3–6 月实习 → 转正率较高 |
| 投递渠道 | 公司官网 / BOSS / 院系内推 |
| 信息时效 | 2026-04（示例数据） |

**面试关注**

- 项目经历的工程化深度（不是仅停留在仿真）
- 是否参与过水下/海上实地测试
- 对硬件、嵌入式、控制三者打通的能力

## 海洋大数据 / AI 类初创企业（示例脱敏）

> **行业方向**：海洋遥感 + AI · **所在地**：上海 / 深圳

| 项目 | 内容 |
| --- | --- |
| 主要岗位 | 算法工程师、海洋数据分析、产品 |
| 学历要求 | 硕士起 |
| 同济对口专业 | 海洋技术、地球科学、AI、统计 |
| 福利 | 期权、灵活办公、行业新方向 |
| 投递渠道 | 公司官网 / 行业群 |
| 信息时效 | 2026-04（示例数据） |

**风险提示**

- 初创公司业务波动较大，建议关注融资轮次与团队背景
- 优先选择有明确商业化场景（如海洋牧场、海上风电、智慧港口）的公司
