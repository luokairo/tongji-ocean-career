## 字段说明

每条企业信息按以下结构记录，便于检索与对比：

| 字段 | 含义 | 示例 |
| --- | --- | --- |
| 单位名称 | 完整全称（含集团/分公司） | 中国船舶集团有限公司第七〇八研究所 |
| 行业方向 | 涉海/工程/AI/装备 等 | 船舶设计 / 海洋装备 |
| 招聘岗位 | 本届开放岗位 | 船舶结构设计工程师、流体力学研究员 |
| 学历要求 | 本/硕/博 | 硕士起 |
| 招聘流程 | 网申/校招/内推 | 9 月网申 → 笔试 → 两轮面试 → offer |
| 联系渠道 | 官网 / HR 邮箱 / 内推人 | hr@example.com（信息源：2024 届学姐） |
| 信息时效 | 最后核实日期 | 2026-04 |
