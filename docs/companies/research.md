# 研究所 / 事业单位

::: info 本页内容为示例模板
研究所与事业单位多为**事业编制**或**预聘-长聘**轨道，与企业招聘节奏差异较大，需早做规划。
:::

## 中科院深海科学与工程研究所（示例）

> **行业方向**：深海技术 · **所在地**：海南三亚

| 项目 | 内容 |
| --- | --- |
| 主要岗位 | 助理研究员、工程师、博士后 |
| 学历要求 | 博士（少量工程岗硕士） |
| 招聘节奏 | 全年招聘 + 春季集中招聘 |
| 投递渠道 | 中科院招聘官网、人才系统 |
| 信息时效 | 2026-04（示例数据） |

**关注要点**

- 关注课题组主页与最新论文方向
- 优秀本科生可申请直博 / 联合培养
- 博后流动站门槛较低，是过渡期不错的选择

## 自然资源部东海局（示例）

> **行业方向**：海洋监测、海洋管理 · **所在地**：上海

| 项目 | 内容 |
| --- | --- |
| 主要岗位 | 海洋调查、监测、数据分析、行政 |
| 学历要求 | 本/硕，部分岗位要求硕士及以上 |
| 招聘节奏 | 每年 11–12 月集中招聘 + 选调专项 |
| 编制 | 事业编 / 参公 |
| 投递渠道 | 单位官网 / 国家海洋局系统招聘 |
| 信息时效 | 2026-04（示例数据） |

**笔试参考**

- 公共基础知识（约 40%）
- 海洋学/海洋技术专业知识（约 60%）
- 公文写作 / 政策理解

## 高校研究院类（示例）

各高校的"高等研究院 / 海洋研究院"近年招聘活跃，路径包括：

- **博士后**（2–4 年，可转编/晋升）
- **预聘副研究员**（6 年长聘考核）
- **工程师序列**

参考单位：上海交大海科院、同济大学海洋高等研究院、华东师大河口海岸研究所、自然资源部第二海洋研究所等。

::: tip
建议在博士期间尽早联系意向导师，了解课题组招聘计划。
:::
