# 求职经验分享

本栏目收录同济海洋专业学长学姐的真实求职经历，覆盖**企业就业、选调、事业单位**三大主流赛道。

::: tip 经验贡献方式
- 👉 [提交 Issue · 经验补充](https://github.com/luokairo/tongji-ocean-career/issues/new?template=experience-submission.yml)（推荐）
- 👉 直接 Fork 仓库新增 Markdown 文件并提 PR
:::

## 赛道速览

| 赛道 | 适合人群 | 准备周期 | 难点 |
| --- | --- | --- | --- |
| **企业就业** | 求职意向明确、追求市场化薪酬 | 大三/研一开始实习 | 笔面试节奏快、信息差大 |
| **选调生** | 偏好体制内、对地区有归属感 | 大三/研一开始备考 | 资格限定严、关注党建履历 |
| **事业单位** | 倾向稳定、对口专业岗位 | 大四/研二准备 | 招考节奏分散、信息分散 |

## 写作模板

为方便阅读与归档，建议提交经验时遵循以下结构：

```markdown
## 基本信息
- 届别 / 学历
- 求职方向（具体单位、岗位）
- 是否同意公开真实姓名

## 求职时间线
（按月份列出关键节点）

## 笔试/面试经历
（具体题型、问题、答题思路）

## 经验总结 / 给学弟学妹的话
（避坑点、可复用的策略）
```

::: warning 隐私提示
- 涉及单位 HR、面试官等第三方信息时，请做好脱敏处理（如"某央企"、"L 同学"）。
- 真实姓名、联系方式建议通过 Issue 提交，由编辑组按贡献者意愿决定是否在站内露出。
:::
