# 项目介绍

**同济海洋就业指南** 是同济大学海洋与地球科学学院的求职经验共享平台，沉淀企业资源、学长学姐经验、简历模板与求职工具，帮助同济海洋学子获取更对称的就业信息。

## 我们做什么

围绕"**校企共建、实践育人**"的目标，通过**企业调研、经验沉淀、就业指导**三大方向持续运营本站：

- 🏢 **企业资源图谱** —— 临港涉海方向的招聘单位汇总，含央企/国企、民企、研究所
- 🎓 **求职经验分享** —— 学长学姐在企业、选调、事业单位等赛道的真实经历
- 📄 **简历专区** —— 海洋/工科方向简历模板与优秀案例点评
- 🗓️ **求职时间线** —— 本科/硕士/博士不同赛道的全年关键节点
- 📊 **毕业去向数据** —— 近 3–5 届师兄师姐的去向分布

## 如何贡献

本站采用 GitHub 开源协作模式，三种贡献方式：

### 1️⃣ 提交 Issue（推荐给非技术同学）

进入 [Issues 页面](https://github.com/luokairo/tongji-ocean-career/issues/new/choose)，选择以下模板之一：

- **📝 经验补充** —— 分享你的求职故事、笔面试经历
- **🏢 企业信息补充** —— 推荐新的招聘单位、岗位信息
- **🐛 站点问题反馈** —— 报告错别字、链接失效、内容错误

维护团队会审核后将内容整理上线。

### 2️⃣ Fork & Pull Request（适合熟悉 Git 的同学）

```bash
git clone https://github.com/luokairo/tongji-ocean-career.git
cd tongji-ocean-career
npm install
npm run docs:dev      # 本地预览
# 修改 docs/ 下的 Markdown 文件
# 提交 PR
```

详见 [CONTRIBUTING.md](https://github.com/luokairo/tongji-ocean-career/blob/main/CONTRIBUTING.md)。

### 3️⃣ 页面底部评论

每一篇文章的最底部都有评论框（基于 Giscus），登录 GitHub 即可发言，适合：

- 简短追问、答疑
- 对已有内容的补充更新
- 经验细节的二次讨论
