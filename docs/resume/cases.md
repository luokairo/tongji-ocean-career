## 案例 1：央企工程师岗简历优化（示例）

### 修改前（典型问题）

```
教育背景
2021.09 - 2024.06  同济大学  海洋技术  硕士

项目经历
1. 海洋数值模拟项目
   参与了项目，做了仿真，写了报告。
2. 实习
   在 XX 公司实习半年。

技能
熟练使用 Word、Excel、PPT
英语 CET-6
```

### 主要问题

- 项目描述**没有具体内容**，无法判断你做了什么
- 实习**没说明岗位、产出**
- 技能段是大学生通用模板，**没有专业含量**
- 缺少**关键词**与目标岗位的关联

### 修改后（结构化 + 量化 + 关键词）

```
教育背景
2021.09 - 2024.06  同济大学  海洋技术  硕士  GPA 3.7/4.0
                  核心课程：流体力学、海洋调查方法、数值方法、机器学习
                  导师：XXX 教授（中国海洋工程学会理事）

项目经历

① 基于 OpenFOAM 的某型 AUV 阻力数值模拟（2023.03 - 2023.12）
  - 项目背景：解决某型 AUV 在 0.5–3 m/s 区间阻力预测偏差>15% 的问题
  - 个人贡献：
    · 独立完成 6 自由度网格剖分（500 万单元），与实测对比误差降至 5%
    · 撰写自动化后处理脚本（Python），将单次分析耗时 8 小时缩至 1.5 小时
  - 成果：相关方法应用于 X 项目验证，作为第二作者发表 SCI 论文 1 篇

② [简化第二个项目，使用同样结构]

实习经历
某央企海洋装备研究院  流体力学实习生（2023.06 - 2023.09）
  - 协助导师完成 X 型号船体的 CFD 仿真验证
  - 独立完成 2 份技术报告，被纳入项目最终交付物
  - 转正考核排名前 30%

技能
- 编程：Python（NumPy/Pandas/Matplotlib）熟练；MATLAB 熟练；C++ 入门
- 仿真软件：OpenFOAM、ANSYS Fluent（独立操作）
- 数据 & GIS：ArcGIS、QGIS（基础）
- 英语：六级 580 / 雅思 6.5；可阅读流体力学英文文献
```

### 点评要点

✅ **量化**：网格规模、误差、耗时缩减、论文数  
✅ **结构化**：每个项目 = 背景 + 贡献 + 成果  
✅ **关键词**：CFD、OpenFOAM、AUV ——精准匹配 JD  
✅ **可信度**：导师、单位、产出可追溯

## 案例 2：跨方向求职 AI 岗（示例）

### 核心策略

海洋背景跨 AI 岗，关键在**搭桥**：

- 突出**海洋数据**项目（自然有 AI 应用）
- 用 GitHub 项目证明工程能力
- 自学 AI 课程要列出**具体课程 + 项目**而非"自学 ML"

### 简历段落示范

```
项目 ① 基于深度学习的海洋表面温度时序预测（2023.06 - 2023.12）
  - 数据：NOAA OISST 1982–2022 月度数据（40 年×1°网格全球）
  - 方法：实现并对比 LSTM / TFT / Informer 三种模型
  - 结果：在 ENSO 强信号期 MAPE 降低 18%，开源代码获 50+ stars
  - 仓库：github.com/username/sst-forecast
```
