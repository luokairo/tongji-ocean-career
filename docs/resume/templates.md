## 推荐写作工具

- **Word / WPS**：上手最快，适合中文求职
- **Overleaf**（在线 LaTeX）：适合科研、海外岗，排版美观稳定
- **超级简历 / Canva**：颜值高，但部分 ATS 系统识别效果差，**仅推荐**用于线下递交、非必需岗位
- **Typst**：新一代排版工具，学习曲线低于 LaTeX

## 写作建议

::: tip 一页 vs 两页
- 应届本科生：**一页**（除非有非常突出的多段经历）
- 研究生：一页（应聘）或两页（学术 / 申博）
- 博士：两页内（应聘）或 CV 不限页数（学术）
:::

## 字体与排版

| 元素 | 中文推荐 | 英文推荐 | 字号 |
| --- | --- | --- | --- |
| 正文 | 思源宋体 / 苹方 / 微软雅黑 | Times / Calibri / Helvetica | 10–11 pt |
| 姓名 | 同正文 | 同正文 | 16–20 pt |
| 标题 | 同正文（加粗） | 同正文（加粗） | 12–13 pt |
| 行距 | 1.15–1.3 | 1.15 | – |
| 页边距 | 上下 1.5cm，左右 1.5–2cm | 同左 | – |

::: warning 切勿用花式字体
- 不要使用"楷体""华文行楷"等装饰字体
- 不要在简历上使用红/紫等强对比色，**深蓝/深灰**最稳
:::

## 文件命名规范

投递时建议命名为：

```
姓名_学校_专业_岗位_yyyymmdd.pdf
张某某_同济大学_海洋技术_流体力学工程师_20260415.pdf
```
