## 简历写作五条铁律

::: tip 通用原则
1. **1 页纸**优先（除非有 3 段以上完整核心经历）
2. 经历描述用 **STAR 结构**：情境 / 任务 / 行动 / 结果
3. **量化结果**：数据、百分比、对比基线
4. **关键词对齐 JD**：HR 系统会做关键词匹配
5. **PDF 投递**：避免 Word 在不同电脑上排版错位
:::

## 海洋方向简历的特殊性

- **科研项目**比"奖项"更打动技术岗 HR：要描述项目目标、技术难点、你的具体贡献
- **出海/实地调研经历**是稀缺亮点，单独列出
- **数据处理 / 编程能力**（Python、MATLAB、GIS、CFD 软件）务必量化
- 中英文简历**双语版本**：央企/事业单位用中文，外企/海外/部分研究所用英文
